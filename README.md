*Materia: Paradigmas y Lenguajes de Programación

**Integrantes**

- Romero Ramírez Emanuel

*Descripción
El trabajo consiste en estudiar y representar el funcionamiento de un sistema de semáforos inteligentes utilizando programación funcional. Para esto, se desarrolló una solución en Common Lisp que permite simular los diferentes estados del semáforo, los cambios entre colores y los tiempos de cada señal, además de obtener información sobre su funcionamiento para
analizar los resultados. Luego de esto se realizó una investigación introductoria sobre el lenguaje OCaml, con el propósito de
comparar sus características con las de Common Lisp y analizar diferentes enfoques para resolver un mismo problema utilizando
herramientas propias de la programación funcional.
